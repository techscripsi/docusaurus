<td>`Timer already started, item(s): [...]`</td>
<td>

요청에는 타이머가 이미 있는 항목에 대한 타이머를 시작하는 지침이 포함되어 있습니다. `started` 상태. 타이머 상태는 변경되지 않은 상태로 유지됩니다. 이것은 오류가 아닙니다. 그러나 이 메시지는 실패한 요청과 함께 나타날 수 있습니다.

:::note
`started` 타이머는 만료될 수 있으며, 만료될 수도 있습니다. `paused` 또는 `stopped` 영원히.
:::

자세한 내용은 다음을 참조하십시오. [아이템 타이머](/docs/concepts/timer) 섹션을 참조하십시오.

</td>
