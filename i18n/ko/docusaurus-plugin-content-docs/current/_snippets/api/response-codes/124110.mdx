<td>`Cannot stop an expired timer, item(s): [...]`</td>
<td>


중지하려는 타이머가 `expired` 상태입니다. `started`, `paused` 또는 `pending` 타이머만 `stopped` 일 수 있습니다.

:::note
타이머의 `expired` 상태는 카트의 수명 주기의 끝입니다.
:::

자세한 내용은 다음을 참조하십시오. [아이템 타이머](/docs/concepts/timer) 섹션을 참조하십시오.

</td>
