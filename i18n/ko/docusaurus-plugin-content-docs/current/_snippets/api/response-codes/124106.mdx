<td>`Cannot pause a stopped timer, item(s): [...]`</td>
<td>

정지하려는 타이머가 `stopped` 상태입니다. 그것은 결코 시작된 적이 없습니다. 에 있는 타이머만 `started` 상태를 일시 중지할 수 있습니다.

:::note
`pending` 타이머는 수동으로 시작할 수 있습니다.
:::
