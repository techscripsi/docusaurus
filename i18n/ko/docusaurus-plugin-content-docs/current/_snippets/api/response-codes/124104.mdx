<td>`Cannot pause a pending timer, item(s): [...]`</td>
<td>

일시 중지하려는 타이머는 `pending` 상태. 그것은 결코 시작된 적이 없습니다. 에 있는 타이머만 `started` 상태를 일시 중지할 수 있습니다.

:::note
`pending` 타이머는 이벤트에 의해 시작될 수 있습니다. (a specified cart state) 또는 수동으로.
:::
