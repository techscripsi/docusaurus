<td>`Timer already stopped, item(s): [...]`</td>
<td>

요청에는 타이머가 이미 있는 항목에 대한 타이머를 중지하라는 지침이 포함되어 있습니다. `stopped` 또는 `expired` 상태. 타이머 상태는 변경되지 않은 상태로 유지됩니다. 이것은 오류가 아닙니다. 그러나 이 메시지는 실패한 요청과 함께 나타날 수 있습니다.

:::note
`stopped` 또는 `expired` 타이머는 다시 시작할 수 없습니다.
:::

</td>
