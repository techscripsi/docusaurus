<h1>옴니카트 기능</h1>

* 올인원 주문: 소비재, 디지털 상품, 일회성 구매 및 구독을 단일 장바구니에 결합
* 유연한 결제 설정: 태그가 적용된 모든 장바구니 항목에 적용되는 태그를 사용하여 결제 설정 구성
* 정밀 조정: 최대 유연성을 위해 장바구니, 태그 또는 항목 수준에서 설정 구성
* 효율적인 API: 더 적은 API 요청과 간단한 시나리오를 위해 설계됨
* 쉬운 캡처 및 환불: 전체 장바구니 또는 개별 항목을 캡처, 취소 또는 환불할 수 있으며, 부분 환불이 지원됨

<h1>체크리스트</h1>

1. 다음과 친숙해지세요 <Link to="/docs/concepts/">핵심 개념</Link>
1. 옴니카트에 등록하고 상인 패널에 대한 테스트 자격 증명을 받으세요
1. 테스트 API 키를 생성하고 이를 사용하여 결제 흐름을 테스트하세요
1. 생산 환경으로 전환하려면 지원팀에 문의하세요
1. 지불 수신을 시작하세요
