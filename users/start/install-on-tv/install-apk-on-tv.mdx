# Инструкция по установке сборки на ТВ через RuStore: от скачивания до запуска

1. Чтобы скачать актуальную версию сборки, отсканируйте QR-код ниже или перейдите по .


2. Перенесите скачанный APK-файл на флешку или своё устройство Android.

3. Вставьте флешку в ваш телевизор.

:::info

- При необходимости отформатируйте флешку.

- Порт USB должен находиться на задней панели телевизора, куда подключаются остальные провода, например, HDMI.

:::

4. Откройте флешку/устройство на ТВ: на телевизоре появится сообщение, что подключено новое устройство. Телевизор автоматически откроет окно для просмотра файлов на флешке или предложит его открыть.

5. Выберите файл и запустите установку. При необходимости предоставьте разрешение на установку.

6. Запустите RuStore с экрана завершения установки или из списка приложений.
