# Страница приложения в RuStore

Выберите приложение в  и нажмите на него.

Откроется окно с информацией о приложении.

*Что можно сделать на странице приложения?*

1. Установить приложение на своё устройство. 




2. Ознакомиться с описанием приложения и узнать о последних обновлениях.



3. Просмотреть рейтинг и прочитать отзывы на приложение.



4. Узнать о возрастных ограничениях и возможном нежелательном контенте в игре.



5. Посмотреть, какие данные запрашивает приложение.



6. Просмотреть похожие игры и приложения.
