# Обновление RuStore: Эффективные шаги для поддержания вашего приложения в актуальном состоянии

При наличии обновлений приложение RuStore отправит вам уведомление с предложением обновить версию приложения.

1. Нажмите **Обновить**.

Отобразится процесс загрузки приложения на ваше устройство. После загрузки появится уведомление.

2. Нажмите **Установить**.


&nbsp;

&nbsp;


Вы можете отложить обновление версии.

Нажмите **Не сейчас** в уведомлении с предложением обновить версию приложения.

:::info

Уведомление покажется повторно через неделю, пока приложение не будет обновлено.

:::
