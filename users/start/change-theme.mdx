# Как легко изменить тему в приложении RuStore

В приложении RuStore доступны следующие темы оформления:

1. Светлая
2. Тёмная
3. Розовая
4. Как в системе

Чтобы изменить тему:

1. Откройте приложение RuStore.
2. В нижнем меню выберите **Моё**. 
3. Выберите пункт **Тема**.
4. Выберите вариант оформления.
5. Нажмите кнопку **Применить**.
