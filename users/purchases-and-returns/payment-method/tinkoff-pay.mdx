# Оплата через : Простой способ оплаты приложений и товаров в RuStore

Вы можете оплатить приложение и товары внутри приложения с помощью .

:::note

Оплата через  доступна только при наличии на устройстве установленного приложения T-Банк.

:::

1. Откройте приложение, в котором хотите оформить подписку или купить товар.
1. Выберите необходимые условия.
1. Нажмите **Оформить подписку**.

  * **Если вы авторизованы через T-ID**

    3.1 Выберите  из списка.<br />
    3.2 Нажмите **Оплатить с **.

  * **Если вы авторизованы другим способом**

    3.1 Выберите  из списка.<br />
    3.2 Нажмите **Оплатить с **.<br />
