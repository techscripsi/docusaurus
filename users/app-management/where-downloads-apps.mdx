# Куда скачиваются приложения из RuStore на Android

На Android расположение загруженных приложений из RuStore стандартизировано. Обычно это папка `Android` — находится во внутренней памяти устройства.

Внутри `Android` две папки:

- `Data` — для хранения данных приложения;
- `Obb` — для кэша игр.

Это закрытые папки, и APK-файлы приложений в них не отображаются.
