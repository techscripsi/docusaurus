# Руководство по установке приложений из RuStore: Полный Гид по Успешному Запуску

Вы можете установить из RuStore: банковские сервисы, игры, соцсети, маркетплейсы. В RuStore добавляются новые приложения, включая топ-20 самых популярных приложений в России.

Перед установкой приложений из RuStore настройте установку приложений из внешних источников.

## Как разрешить установку из внешних источников {#setup}

1. Перейдите в настройки устройства.
2. Выберите раздел **Безопасность**.
3. Найдите **Дополнительные настройки**.
4. Выберите **Установка приложений из неизвестных источников**.
5. Найдите магазин RuStore и нажмите на него.
6. Установите переключатель в активное положение **Разрешить установку приложений**.



Вы также можете перейти  устройства из самого RuStore:

1. Выберите приложение из каталога RuStore.
2. Нажмите **Скачать**.
3. Во всплывающем окне нажмите кнопку .
4. Повторите шаги 2 — 6 из  выше.



:::note

Авторизуйтесь в RuStore, чтобы устанавливать платные приложения. 

:::

Если вы не авторизованы, вы сможете установить только бесплатные приложения.

## Как установить приложение

1. Откройте приложение RuStore.
2. Выберите в  приложение и нажмите на него.
3. Нажмите **Скачать**. 

  

  На иконке приложения отобразится процесс загрузки.

  После загрузки появится уведомление, запрашивающее разрешение на установку.

4. Нажмите **Установить**.

## Как установить ранее удаленное приложение

, чтобы просматривать список установленных на устройстве и доступных к обновлению приложений в разделе **Моё**.

1. Откройте приложение RuStore.
2. В нижнем меню выберите **Моё**. 

 
3. Нажмите **Все приложения** и выберите **Нет на устройстве**.

 
4. Выберите приложение из списка и нажмите **Скачать**. 

:::note
Через вкладку **Нет на устройстве** также можно найти и установить приложения, которые ранее были установлены под учётной записью RuStore на других устройствах.
:::
