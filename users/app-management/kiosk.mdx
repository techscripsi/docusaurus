# Что такое «Киоск»

**Киоск** — это витрина цифровых товаров, через которую можно пополнить баланс для внутриигровых покупок.

:::note

Обновите RuStore до последней версии, чтобы начать пользоваться сервисом.

:::

## Как пополнить баланс

1. Откройте приложение RuStore на своём мобильном устройстве.
2. В панели внизу выберите вкладку **Киоск**.

:::note

Сейчас доступна оплата только через **.

:::

5. Нажмите **Оплатить**.

На указанный адрес электронной почты будет отправлен ваучер вместе с инструкцией по активации.

Готово! Вы можете использовать ваучер, чтобы пополнить игровой кошелёк.

:::warning

Возврат средств за ваучер после покупки недоступен. Если у вас возникли сложности с активацией, обратитесь в поддержку: help@foreignpay.ru

:::
