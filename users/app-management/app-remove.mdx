# Как удалить приложение из RuStore


, чтобы просматривать список установленных на устройстве приложений в разделе **Моё**.

1. Откройте приложение RuStore.
2. В нижнем меню выберите **Моё**. 

 

3. Найдите приложение, которое вы хотите удалить, и перейдите на страницу этого приложения.

4. В правом верхнем углу нажмите 

 

5. Во всплывающем окне нажмите **ОК**, чтобы подтвердить удаление.

Приложение будет удалено с вашего устройства.
