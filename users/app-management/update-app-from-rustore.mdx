# Как обновить приложения в RuStore

Вы можете обновить любое установленное на устройстве приложение, для которого в RuStore доступна более новая версия. Для этого необязательно авторизовываться в RuStore.

:::note
Полный список доступных к обновлению приложений доступен для  пользователей в разделе **Моё**.
:::

1. Откройте приложение RuStore.
2. В нижнем меню выберите **Моё**. 

 
3. (Опционально) Нажмите **Обновить всё**, если хотите обновить все доступные для обновления приложения.
4. Перейдите в раздел **Все приложения** и выберите **На этом устройстве**.

 
4. Выберите приложение из списка и нажмите **Обновить**.
