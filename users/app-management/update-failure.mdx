# Руководство по исправлению ошибок обновления приложений в RuStore

В процессе обновления могут возникать ошибки обновления и сбои. В большинстве случаев ошибки типовые:

1. Подключение устройства к интернету.
2. Наличие свободной памяти на устройстве.
3. Запрещена установка из неизвестных источников.
4. Версия Android.

Если не удалось обновить приложение:

1. Отключитесь и снова подключитесь к Wi-Fi сети или мобильному интернету.
2. Удалите лишние данные во внутреннем хранилище и на microSD карте, если место закончилось.
3. В настройках устройства разрешите устанавливать неизвестные приложения.
4. Обновите Android до последней версии. Для установки некоторых приложений имеет значение версия Android.

Одна из причин сбоя обновления приложения — несовместимость версий. Она может возникнуть, если вы установили на свое устройство приложение из другого магазина.

Удалите текущую версию и повторите установку приложения из RuStore, чтобы обновить приложение.
