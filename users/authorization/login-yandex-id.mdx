# Как авторизоваться в RuStore через Яндекс

Вы можете авторизоваться в RuStore через Яндекс:

5. Проверьте актуальность номера телефона.
6. Нажмите **Подтвердить**.

На ваш номер придет SMS с кодом подтверждения.

1. Введите код.
2. Нажмите **Продолжить**.
3. Введите пароль от своего VK ID.
4. Нажмите **Продолжить как**.
