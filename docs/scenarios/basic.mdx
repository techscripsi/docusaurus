## Two-Step Payment {#twostep}

<Mermaid
  value={`
    sequenceDiagram
      autonumber
        participant C as Customer
        participant M as Merchant
        participant O as OmniCart
        participant V as Verification Service
        C ->> M: customer begins payment
        M ->>+O: init request
        O -->>-M: init response
        M ->> C: invoice delivery
        C ->>+V: payment authorization
        V ->>-O: authorized
      activate O
        O ->>-O: funds on hold
        M -->> C: success message
        M ->>+O: capture request
        O ->> O: payment
        O -->>-M: capture response
      opt
        M ->>+O: status request
        O -->>-M: status response
      end
