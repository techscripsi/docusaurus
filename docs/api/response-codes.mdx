# Response Codes and Messages


they might sometimes overlap with HTTP codes but it is consistent

```
x   xx   xxx
│   │     └─ specific error code
│   └─ error category and subcategory
│      ├─ 0 - related to authentication
│      │  ├─ 00 - authentication successful
│      │  └─ 01 - authentication failed
│      ├─ 1 - data related
│      │  ├─ 10 - no errors
│      │  ├─ 11 - did not receive expected data
│      │  ├─ 12 - the received data has wrong format
│      │  ├─ 13 - the received data has invalid value
│      │  ├─ 14 - conflict with existing data
│      │  └─ 19 - other data related errors
│      ├─ 2 - permissions
│      │  ├─ 20 - no errors
│      │  ├─ 21 - not allowed to pass this method
│      │  ├─ 22 - not allowed to pass this object or parameter
│      │  ├─ 23 - the passed value is out of the allowed range
│      │  └─ 29 - other permission related errors
│      └─ 9 - other cases
└─ is the request successful
   ├─ 0 - success
   └─ 1 - failure
```

- 400 Bad Request
- 405 Method Not Allowed
- 409 Conflict
